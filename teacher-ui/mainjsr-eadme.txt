# 🐾 白貓工作室 教師端 UI 說明文件（teacher-ui）

## 📘 目的
本模組提供教師即時監控學生作答情況，包含：
- 自動亮紅燈提示陌生作答者
- 顯示手寫圖像作答題號資訊
- 支援 Firebase Realtime Database 串接

---

## 🗂 結構說明

| 檔案名稱 | 說明 |
|--|--|
| `index.html` | 教師介面主畫面（左：控制區、右：回應區） |
| `main.js` | 監聽 Firebase `/handwriting/guest` 並即時更新畫面、顯示紅燈與警示 |
| `style.css` | 定義畫面排版、紅色動畫樣式（blingbling） |
| `firebase-config.js` | Firebase 連線設定（如已整合入 main.js 可省略） |
| `cat.png` | 可做為裝飾圖、暫未啟用 QR 圖示 |

---

## 🔥 核心功能

- 監聽：`handwriting/guest/{questionId}`
- 判斷是否為陌生（`student-row` 中找不到 guest）
- 加入：紅色名單標籤、response 區顯示作答題號與警告文字
- 動畫：`.response-box.red` 使用 `@keyframes bling` 動畫閃爍

---

## ✅ 畫面區域說明

### 左側（控制欄）
- Take Screenshot（未啟用）：預留擷圖功能位
- QR 圖片區（預設無圖）：可動態產生或留空
- LNK：複製登入連結
- ASK：預留教師出題介面入口
- 學生名單（A–F）＋燈號：預設常駐

### 右側（回應板）
- 即時學生回答狀況會顯示於此（包括文字與陌生紅框）
- `.response-board` 區域為訊息顯示主區域

---

## 🧪 測試流程建議

1. 從學生端上傳 `/handwriting/guest/自由繪圖_123` 一筆資料
2. 教師端會自動新增紅燈名單 `guest（陌生）`
3. 回應板出現 `⚠️ 陌生學生 guest：手寫圖作答於「自由繪圖_123」`
4. 邊框閃爍動畫觸發

---

## 📌 注意事項

- 若 QR 圖片沒上傳，不影響功能
- 若 Firebase 無 `guest` 寫入，畫面不會變化屬正常
- 建議先開放 Firebase rules 為：

```json
{
  "rules": {
    "handwriting": {
      ".read": true,
      ".write": true
    }
  }
}
```
